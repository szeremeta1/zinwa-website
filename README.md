# Zinwa Technologies Website

A modern, BlackBerry-style website for Zinwa Technologies based in Xiamen, China.

## Overview

This website showcases the Zinwa Q25 smartphone, a modified BlackBerry Q20 Classic featuring a physical QWERTY keyboard and modern specifications. The design is inspired by BlackBerry's clean, minimalistic aesthetic while maintaining exclusive Zinwa branding.

## Features

- **BlackBerry-Inspired Design**: Clean, professional black and white color scheme
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Modern Animations**: Smooth transitions and scroll-based animations
- **Interactive Elements**: Hover states, navigation, and form handling
- **Physical Keyboard Showcase**: Animated phone mockup highlighting the Q25's QWERTY keyboard

## Local Development

To view the website locally:

1. Clone the repository
2. Open `index.html` in a web browser, or
3. Run a local server:
   ```bash
   python3 -m http.server 8000
   ```
4. Navigate to `http://localhost:8000`

## Design Principles

- Minimalistic and clean layout
- Professional black and white color palette
- Sharp geometric elements
- Smooth animations and transitions
- Mobile-first responsive design
- No BlackBerry branding (Zinwa branding only)
